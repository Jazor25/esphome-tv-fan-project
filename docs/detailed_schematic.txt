Ground Distribution:
==================
                            Common Ground Bus
    ┌─────────────────────────────────────────────────────────────┐
    │                                                             │
    ├─ 12V Power Supply (-)                                       │

    ├─ DHT22 #1 (Pin 4)                                           │
    ├─ DHT22 #2 (Pin 4)                                           │
    ├─ FET Board #1 (GND)                                         │
    └─ FET Board #2 (GND)                                         │

Control Logic:
=============
Temperature Sensor 1 (GPIO22) → Controls → Fan 1 (GPIO32)
Temperature Sensor 2 (GPIO21) → Controls → Fan 2 (GPIO25)

Default Settings:
- Fan ON Temperature: 31.0°C
- Fan OFF Temperature: 29.0°C (Fan 1), 28.0°C (Fan 2)
- Check Interval: 10 seconds
- Sensor Update: 20 seconds

Signal Flow:
===========
DHT22 → ESP32 → Temperature Reading → Logic → GPIO Output → FET Board → Fan Control

Power Flow:
==========
12V Supply → LM2596s → 3.3V → ESP32
12V Supply → FET Boards → Fans (when triggered)
ESP32 3.3V → DHT22 Sensors
